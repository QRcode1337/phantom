The ARGUS Feed view for the Phantom anomaly detection engine.

**DESIGN SYSTEM (REQUIRED):**
*   **Theme:** "Tactical Glass OS". Deep space gradient background (dark purple to absolute charcoal). UI elements are heavily blurred glass panels (`bg-white/5`, `backdrop-blur-2xl`) with ultra-thin white borders and large rounded corners (`2.5rem`).
*   **Typography:** Floating, precise typography. Utilize ultra-light "hairline" weights (`100`) for both `Inter` (sans-serif UI) and `JetBrains Mono` (Data/Numbers). Use extremely wide letter-spacing (`tracking-widest`) for small uppercase labels.
*   **Color Palette:** Pure white text (often semi-transparent at 40% or 60%). Neon glowing accents: Aurora Green (`#00ff88`) for system status/positive edge, Solar Yellow (`#ffbb00`) for warnings, Alert Red (`#ff3333`) or Hot Pink (`#ff00ff`) for critical anomalies.
*   **Vibe & Visuals (CRITICAL):** Highly advanced, ambient, abstract military OS. Centerpiece data visualizations MUST be extremely complex abstract 3D vector drawings/SVGs. Imagine an isometric view of a Strange Attractor, overlapping twisted 3D ribbons with gradient strokes, or deep layered node constellation meshes. Use transparency, drop shadows, and overlapping vector elements to create deep 3D complexity.

**Page Structure:**
1.  **Header Navigation:**
    *   Logo: `PHANTOM // ARGUS` (Ultra-light font, wide tracking).
    *   System Status: A glowing Alert Red dot (pulsing indicator).
    *   Right Icon: `radar` icon in a glass bubble.
2.  **Top Metrics Grid (3 glass cards):**
    *   Seismic Nodes (e.g., Active).
    *   Flight Tracks (e.g., 2,410).
    *   Critical Anomalies (e.g., 1 detected).
    *   *Design note: Use hairline font weight for numbers.*
3.  **Main Content Section (Large Glass Panel taking up 50% of the screen):**
    *   **Header:** "Geospatial Anomaly Mesh".
    *   **Expanded 3D Visualization:** Generate a highly complex 3D vector (SVG) visualization representing a global radar mesh or geospatial anomaly web. It should look like an intricately connected node constellation or a wireframe globe, glowing intensely with Hot Pink and Alert Red where anomalies cluster, fading into dark transparency elsewhere. Use overlapping glowing SVG paths and scatter plots with different opacities to create immense 3D depth and complexity.
4.  **Anomaly List (Below the visualization, separated by thin borders):**
    *   A vertically stacked stream of live geospatial anomalies (flight deviations and seismic events).
    *   Row 1 (Critical): Alert Red ambient glow. "FLIGHT DEVIATION: ICAO 0xAFF4", "Altitude Chaos: 0.88", "Severity: Critical".
    *   Row 2 (Warning): Solar Yellow ambient glow. "SEISMIC REGIME: CLUSTER_B", "Magnitude Shift: 4.2", "Severity: Medium".
    *   Row 3: Dim grey. "NODE SYNC", "System Nominal".
5.  **Bottom Navigation:**
    *   Sticky glass pill at the bottom with 4 icons: Grid (Overview), Cloud (Weather), Show_Chart (Price), Radar (Argus - Active).
